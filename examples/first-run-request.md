使用 stitch-frontend-workflow skill。

我要做一个中文 AI 工作流控制台页面，主题是 Gemini、Stitch 和 Codex 如何协同完成前端 UI。

请按这个流程：
1. 先阅读当前项目结构。
2. 创建一个本地 HTML 预览，放到 stitch-previews/ai-workflow-console/。
3. 如果 Stitch MCP 可用，用同一份 brief 在 Stitch 里生成一个 UI screen 或变体。
4. 读取 Stitch 返回的 HTML / 截图 / 设计上下文。
5. 选择一个方向，说明理由。
6. 再把它落到当前前端项目里。
7. 最后用浏览器截图验证桌面和移动端。
