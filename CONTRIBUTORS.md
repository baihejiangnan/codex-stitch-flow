## Note

GitHub 仓库页面里的自动 Contributors 列表通常根据 commit 作者统计。  
如果希望 Codex 出现在提交记录中，可以在相关 commit message 中加入类似：

```text
Co-authored-by: OpenAI Codex <codex@openai.com>
```

这不保证映射到一个 GitHub 用户账号，但可以在提交文本里保留 AI 协作记录。
